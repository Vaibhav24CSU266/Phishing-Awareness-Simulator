# Phishing Awareness Simulator

## Overview
This project is designed to educate users about phishing attacks and help them identify suspicious emails.

## Features
- Email phishing examples
- Legitimate vs phishing email comparison
- Awareness quiz
- Security tips

## Sample Phishing Indicators
- Suspicious sender address
- Urgent language
- Unexpected attachments
- Fake login links
- Requests for sensitive information

## Awareness Tips
1. Verify sender identity.
2. Do not click unknown links.
3. Check URLs carefully.
4. Enable MFA.
5. Report suspicious emails.

## Educational Purpose
This project is intended for cybersecurity awareness and training purposes only.
